# Linux File Permissions Portfolio Activity

##  Description

This repository contains a portfolio activity demonstrating the use of Linux commands
to review and manage file and directory permissions. The project focuses on improving
system security by ensuring proper authorization levels using `ls` and `chmod`.

This activity is part of a cybersecurity / IT support learning program.

---

##  Skills Demonstrated

- Linux file and directory permission management
- Understanding permission strings (`rwx`)
- Using `ls -la` to review permissions
- Using `chmod` to modify permissions
- Basic system security principles
- Technical documentation
